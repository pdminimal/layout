## Tool stack
### Frontend
- [TypeScript](https://www.typescriptlang.org/)
- [Node.js](https://nodejs.org/)

- [Vite](https://vitejs.dev/) for HMR etc.
- [Vitest](https://vitest.dev/) for unit testing
- [Rome formatter](https://rome.tools/) for formatting
